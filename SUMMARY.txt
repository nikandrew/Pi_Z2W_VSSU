🐛 РЕШЕНИЕ ПРОБЛЕМ:
═════════════════════════════════════════════════════════════════════════════

Проблема: UART порт не открывается
Решение:
  1. Проверьте доступный порт: ls -la /dev/serial* /dev/ttyAMA*
  2. Активируйте UART: sudo raspi-config → Interface Options → Serial Port
  3. Перезагрузитесь: sudo reboot

Проблема: GPIO17 выдает ошибку
Решение:
  1. Запустите с sudo: sudo python3 main.py
  2. Проверьте, что GPIO17 не используется другой программой

Проблема: Камера не записывает
Решение:
  1. Проверьте камеру: rpicam-vid -t 1000 -o test.h264
  2. Активируйте камеру: sudo raspi-config → Interface Options → Camera
  3. Перезагрузитесь

Проблема: RS-485 не принимает данные
Решение:
  1. Проверьте скорость: должна быть 115200 бод
  2. Проверьте подключение кабелей (A, B, GND)
  3. Используйте oscill для отладки сигналов
  4. Запустите тест: sudo python3 test_rs485.py
