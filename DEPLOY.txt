════════════════════════════════════════════════════════════════════════

На Pi:

  # Создайте systemd сервис
  sudo nano /etc/systemd/system/camera.service
  
  Содержимое:
  ─────────────────────────────────────────────────────────
  [Unit]
  Description=Camera Recording System
  After=network.target
  
  [Service]
  Type=simple
  User=pi
  WorkingDirectory=/home/pi/CODE/Pi_Z2W_VSSU
  ExecStart=/usr/bin/python3 /home/pi/CODE/Pi_Z2W_VSSU/main.py
  Restart=on-failure
  RestartSec=10
  
  [Install]
  WantedBy=multi-user.target
  ─────────────────────────────────────────────────────────
  
  # Активируйте сервис
  sudo systemctl enable camera.service
  sudo systemctl start camera.service
  
  # Проверьте статус
  sudo systemctl status camera.service

Проблема: connection refused на SSH
  Решение: Убедитесь, что Pi подключена в сеть и включена

Проблема: rpicam-vid не найден
  Решение: sudo apt install -y rpicam-apps

Проблема: GPIO17 выдает Permission denied
  Решение: Запустите с sudo: sudo python3 main.py

Проблема: UART не открывается
  Решение: 
    1. Проверьте: ls -la /dev/serial0
    2. Активируйте: sudo raspi-config → Interface Options → Serial Port
    3. Перезагрузитесь

Проблема: Камера не работает
  Решение:
    1. Проверьте подключение CSI кабеля
    2. Активируйте: sudo raspi-config → Interface Options → Camera
    3. Перезагрузитесь
    4. Тест: rpicam-vid -t 5000 -o test.h264
