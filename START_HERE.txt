🔄 ПОТОК ВЫПОЛНЕНИЯ:
═════════════════════════════════════════════════════════════════════════════

  t=0 сек    → Получен кадр 00 01 + CRC-32/ISO-HDLC
  t=0.1 сек  → Запуск записи видео (rpicam-vid)
  t=10 сек   → Видео завершено
  t=10-20 сек → Разбиение на части (ffmpeg)
  t=20 сек   → Отправка ответа 'recording_complete'
  
  Результат:
    ✓ videos/video_YYYYMMDD_HHMMSS.h264  (15-20 МБ)
    ✓ Ответ отправлен по RS-485

💡 ИСПОЛЬЗОВАННЫЕ ТЕХНОЛОГИИ:
═════════════════════════════════════════════════════════════════════════════

Python (основной язык):
  • asyncio - асинхронное программирование
  • subprocess - запуск системных команд
  • logging - логирование
  • pathlib - работа с файлами
  • serial - RS-485 коммуникация
  • gpiozero - управление GPIO

Системные инструменты:
  • rpicam-vid - запись видео
  • ffmpeg - разбиение видео
  • bash - скрипты установки

ОС:
  • Raspberry Pi OS (64-bit)
  • Python 3.8+

Оборудование:
  • Raspberry Pi Zero 2W
  • Camera Module 3 (Sony IMX708)
  • SN65HVD1781AQDRQ1 RS-485
